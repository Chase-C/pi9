# Pi9

Pi9 is a collection of plugins for the pi coding agent. On contact with an ordinary pi instance, pi9 acts as a seed crystal, causing pi to undergo a phase transition into a more capable harness.

Extensions:
- @pi9/subagent: An extension that lets pi delegate to subagents, show live progress, and manage process-lifetime subagent sessions.
- @pi9/whisper: An extension that allows two pi agents to communicate with each other, sharing information and collaborating on tasks.
